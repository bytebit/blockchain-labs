哈希函数与数据完整性验证实验
【实验目的】
1. 理解密码哈希函数基本性质（单向性、抗碰撞性、固定长度输出）
2. 掌握SHA256哈希算法的基本使用
3. 理解区块链中数据完整性验证机制

【实验原理】
哈希函数可以将任意长度输入映射为固定长度输出。区块链利用哈希函数对区块数据进行摘要计算，从而实现数据完整性检测。

【实验步骤】
步骤1：编写SHA256计算程序
步骤2：实现文件完整性检测
步骤3：验证哈希雪崩效应

【参考代码】
import hashlib

def sha256_hash(data):
    sha = hashlib.sha256()
    sha.update(data.encode())
    return sha.hexdigest()

msg = "blockchain security"
print("Hash:", sha256_hash(msg))
