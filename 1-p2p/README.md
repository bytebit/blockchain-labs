基于比特币 P2P 网络原理的 Go 语言演示程序。

## 功能特性

- 节点发现：类似比特币的 addr 消息，节点之间互相分享已知节点地址
- 握手协议：节点连接时进行版本握手
- 地址广播：每 10 秒向所有连接的节点广播已知节点地址
- 自动连接：发现新节点地址后自动尝试连接
- 定期打印：每 5 秒打印出当前发现的所有节点地址

## 使用方法

### 快速测试（Windows）

直接运行 `test_nodes.bat`，它会自动启动 4 个节点并互相连接。

### 手动启动节点

**启动第一个节点（种子节点）：**
```bash
go run main.go -addr 127.0.0.1:8001
```

**启动第二个节点，连接到种子节点：**
```bash
go run main.go -addr 127.0.0.1:8002 -seeds 127.0.0.1:8001
```

**启动更多节点：**
```bash
go run main.go -addr 127.0.0.1:8003 -seeds 127.0.0.1:8001
```

## 命令行参数

- `-addr`：节点监听地址（默认：127.0.0.1:8000）
- `-seeds`：逗号分隔的种子节点地址列表

## 问题修复记录

### 问题：出现过多临时端口节点

**原因**：代码错误地将 TCP 连接的临时源端口也当成了节点地址。

**解决方案**：移除了在 `handleConnection` 函数中错误添加临时端口的代码，只通过握手消息和 addr 消息获取有效的节点地址。
